# Page Insight Chrome Extension (Prototype)

## What it does:
- Shows page title, URL, number of images, number of links
- Buttons to highlight all links on the page
- Button to darken the page for focus

## How to test (Desktop):
1. Unzip this folder to a location on your computer
2. Open Google Chrome
3. Go to: `chrome://extensions/`
4. Enable "Developer Mode" (top right)
5. Click "Load unpacked" and select this unzipped folder
6. Click the extension icon in Chrome toolbar to use it

## Notes:
- This is a prototype, fully functional for testing and demonstration
- Works on any webpage
